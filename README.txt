This project contains solutions to various problems from Project Euler (see projecteuler.net)

Read the wiki on how to run the program for a specific problem.

Directory structure of the project is as follows:
/include has user-defined header files
/lib has source code for user-defined libraries
/src has the source code for the main entry point of the application
/res contains miscelleneous resources, e.g. a text file with the 20x20 grid input for problem 11
